Нам нужно написать бэкенд для сервиса аренды автомобилией

Для этого нам нужно 3 сущности
1) Пользователь
2) Автомобиль
3) История аренды

1. Пользователь: поля продумайте самостоятельно, но важно чтобы пользователь мог быть верифицированым (подтвержденное в.у.)(
верификация - это просто отдельный эндпоинт, который выставит юзеру состояние "верифаед", и он сможет отправлять запрос за "взять в аренду")
2. Автомобиль: ну тут +\- все просто
3. История аренды: автомобиль можно взять в аренду (посуточно, если берем автомобиль в аренду, то делаем запись в "историю аренды" с указание дат, на которые автомобиль в аренде. 
p.s. аренда может быть отменена 


Пользователи: (CRUD)+верификация
История аренды: (CRD)
Авто: (R + CUD[по токену])
